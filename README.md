# Out at the Fair® V6.6

V6.6 is the equal-destination, performance, SEO and accessibility release.

## Network changes
- Removed the Marin destination page entirely
- Removed Marin from current fair navigation, cards and map
- Preserved Marin only in the historical timeline
- Current fairs now appear equally at once
- Removed mobile horizontal swiping for destination cards
- One-column phone layout, two-column tablet layout and three-column desktop layout
- Added clear language that no current fair is featured above another

## Mobile and layout improvements
- Every fair is visible without swiping
- Equal card dimensions and visual weight
- Improved card content alignment
- Better line wrapping and consistent phone gutters
- Preserved the V6.5 hamburger menu fix

## Performance
- Added content-visibility to long-page card sections
- Improved image lazy loading and decoding
- Added image dimensions where possible to reduce layout shift
- Updated the service-worker cache
- Removed the deleted Marin page from offline caching

## Accessibility
- Stronger keyboard focus rings
- Improved skip-link behavior
- 44px minimum interaction targets
- Better muted-text contrast
- Reduced-motion support
- Improved landmark labeling
- External-link security attributes

## SEO
- Added Open Graph defaults
- Added Twitter card metadata
- Added robots.txt
- Added sitemap.xml
- Preserved existing structured data and canonical URLs

## Deployment
Upload the contents of this ZIP directly to the root of the staging GitHub repository.
No CNAME file is included.
